# auto-bot
Arduino based simple robot controlled by Bluetooth.

## Parts list
- Arduino MEGA
- L293D Motor Control Shield v1.0
- MINI DC GearBox x4
- HC-05 Bluetooth Module
- HC-SR04 Sensor
- Protoboard

## Credits

### Used libraries and other 3rd party code
